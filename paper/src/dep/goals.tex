Common branch problems that we're trying to solve

\begin{enumerate}
    \item Evaluate $p(\theta \mid x)$
    \item Sample $p(\theta \mid x)$
    \item $\mathbb{E}[\theta \mid x], \mathbb{D}[\theta \mid x]$, quantiles
    \item $\theta_{MLE} = argmax_{\theta \sim p(\theta)} p(x \mid \theta)$
    \item $\theta_{MAP} = argmax_{\theta \sim p(\theta)} p(\theta \mid x)$
\end{enumerate}
